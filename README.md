# 🏠 房价预测项目

基于深度学习的房价预测模型，《动手学深度学习》这本书的“4.10 实战Kaggle比赛：预测房价”实现以及代码解析。


## 📖 项目简介

本项目使用深度学习技术预测爱荷华州埃姆斯的住宅房价。通过特征工程、神经网络建模和超参数调优，仅限于以后方便查看和学习。

**比赛链接**: [House Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques)


### 核心技术
- **深度学习**: PyTorch
- **数据处理**: Pandas, NumPy
- **可视化**: Matplotlib
